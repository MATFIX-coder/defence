# Python-задачи: Апрельская и Майская защита

Локальный учебный сайт для подготовки к защите по программированию.

## Запуск

Откройте файл `index.html` в браузере.

## Что внутри

- Апрельская защита – базовые задачи.
- Апрельская защита – задачи поинтереснее.
- Майская защита – новые задачи: product, permutations, combinations, функции, координаты, строки, списки, словари, матрицы.

## Важно

Во вкладке «Майская защита» используется отдельное изображение `assets/may-defense.png`.

Проверка полного квадрата оформлена в стиле:

```python
def kv(x):
    return int(x ** 0.5) ** 2 == x
```

Проверка простого числа оформлена в стиле:

```python
def prost(x):
    return x != 1 and all(x % d != 0 for d in range(2, x))
```


## Стиль решений

В решениях Майской защиты добавлены комментарии: номер задачи, краткое условие и ответ перед кодом.

## Режим ученика / сотрудника

По умолчанию сайт открывается в режиме ученика: ответы и решения скрыты.
Чтобы открыть ответы и решения, введите код сотрудника:

```text
E2026
```

Важно: это статический локальный сайт. Код закрывает ответы в интерфейсе, но не является криптографической защитой от просмотра исходных файлов.
